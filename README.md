# Homebrew Tap — amiwrpremium

Personal Homebrew tap. Formulae here are auto-updated by [GoReleaser](https://goreleaser.com) when their source repos cut a release.

## Install

```sh
brew tap amiwrpremium/tap
brew install <formula>
```

Or in one shot:

```sh
brew install amiwrpremium/tap/<formula>
```
